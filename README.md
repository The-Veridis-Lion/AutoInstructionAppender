# 🧩 插件简介
一个用于在发送消息时自动附加指令、且支持完全隐身发送的酒馆插件。

# ✨ 核心功能
- 多指令管理：支持新增、编辑、删除、启用/禁用多条指令。
- 最高权重注入：在发送前自动拼接已启用指令，减少漏加系统约束。
- 无痕隐身模式（HTML注释物理注入）：将指令以注释形式附加到用户输入中，不影响常规阅读。
- 历史记录大清洗：发送前对历史用户消息执行旧指令块清理，避免脏数据累计。
