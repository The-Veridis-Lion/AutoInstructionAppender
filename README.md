# 🧩 插件简介
一个用于在发送消息时自动附加指令、且支持完全隐身发送的酒馆插件。

# ✨ 核心功能
- **多指令管理**：支持新增、编辑、删除、启用/禁用多条指令。
- **角色联动 (锁链模式)**：指令可绑定特定角色。切入该角色时自动勾选开启，切出时自动取消勾选。
- **全局控制 (星号模式)**：设为默认指令后，其开关状态不受切换角色影响，完全由手动控制。
- **最高权重注入**：在发送前自动拼接已启用指令，减少漏加系统约束。
- **无痕隐身模式**：将指令以 HTML 注释形式附加到用户输入中，不影响常规阅读。
- **历史记录大清洗**：发送前自动清理历史消息中的旧指令块，防止数据堆积。

# 📖 简易使用指南
为了让你实现“无感”自动化，请根据需求设置指令右侧的图标：

1.  **想让指令“只在某个角色”生效**：
    * 点亮 **“锁链”** 🔗：进聊天自动开，出聊天自动关。
2.  **想让指令“全局通用”**：
    * 点亮 **“星号”** ⭐：手动档。你打开它就一直开，你关掉它就一直关，切换角色不影响它。
3.  **既没点锁链也没点星号**：
    * 被视为临时指令。只要你切换角色，它就会自动取消勾选关闭。
